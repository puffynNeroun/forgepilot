# ForgePilot MVP Scope

## MVP goal

Ship a small but real dashboard that proves Project Forge can guide production-minded AI-assisted development.

## In scope

- one local product/project workspace
- editable product spec sections
- task lifecycle board
- decision log
- dogfooding log
- handoff prompt generator
- release timeline
- seed/demo data if needed
- clear README and local setup instructions

## Out of scope

- auth
- teams
- billing
- GitHub OAuth
- automatic GitHub import
- background jobs
- real AI agent execution
- multi-product SaaS architecture
